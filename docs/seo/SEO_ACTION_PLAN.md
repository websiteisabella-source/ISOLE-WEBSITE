# Plan SEO para ISOLÉ

Este documento convierte el sitio actual de ISOLÉ en una base más clara para Google y para personas que buscan comprar ropa femenina.

## Prioridad 1: indexación técnica

- Configurar `NEXT_PUBLIC_SITE_URL` con el dominio real antes de publicar. El sitemap, los canonical y los datos estructurados dependen de ese valor.
- Publicar y enviar `https://TU-DOMINIO/sitemap.xml` en Google Search Console.
- Verificar en Search Console las URLs principales: home, `/product/vestido-atardecer`, `/product/blusa-seda-alba` y `/product/slip-petalo`.
- Revisar que `robots.txt` permita rastreo y apunte al sitemap.

## Prioridad 2: búsquedas de compra

Atacar búsquedas con intención comercial y lenguaje natural, sin forzar palabras clave:

- comprar ropa femenina elegante
- vestidos de lino para mujer
- blusas de seda para mujer
- vestidos románticos para mujer
- ropa natural para mujer
- showroom de ropa femenina
- ropa femenina en lino y seda

Cada producto debe responder de forma explícita:

- Qué es la prenda.
- Para quién o para qué ocasión funciona.
- Material, color, fit y sensación al usarla.
- Cómo comprar o consultar disponibilidad.

## Prioridad 3: datos de producto

Ya se agregó `Product` structured data con nombre, descripción, marca, categoría, material, color, imágenes y URL.

Para ser más fuerte en resultados de compra de Google faltan datos comerciales reales:

- Precio.
- Moneda.
- Stock.
- SKU.
- Tallas.
- Política de envíos.
- Política de devoluciones.

Cuando esos datos estén definidos, conviene añadir `Offer`, `shippingDetails` y `hasMerchantReturnPolicy`.

## Prioridad 4: contenido editorial

Crear páginas o secciones útiles que respondan dudas reales antes de comprar:

- Guía para elegir vestidos de lino.
- Como cuidar prendas de seda.
- Qué usar para una tarde o evento cálido.
- Diferencias entre lino, seda lavada, viscosa y satén.
- Como agendar una visita al showroom.

Estas piezas deben enlazar a productos relevantes y a WhatsApp.

## Prioridad 5: medición

Medir semanalmente:

- Consultas que activan impresiones.
- Productos con clics.
- Páginas indexadas.
- Errores de datos estructurados.
- Core Web Vitals.

SEO no garantiza primeros lugares inmediatos; Google indica que los cambios pueden tardar desde horas hasta meses. La meta es mejorar rastreo, comprensión, confianza y conversión de forma sostenida.
